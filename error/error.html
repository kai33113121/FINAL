Error #8: Cierre Inesperado al Exportar Facturas

Descripción del Error

El sistema se cierra inesperadamente al exportar más de 100 facturas a Excel sin mostrar mensaje de error.

Tipo de Error

Rendimiento/Estabilidad

Pasos para Reproducir el Error:

  1. Ir al módulo de consulta de facturas
  2. Seleccionar un rango que incluya más de 100 facturas
  3. Hacer clic en "Exportar a Excel"
  4. Esperar el proceso de exportación

Resultado Esperado

El sistema debe exportar todas las facturas seleccionadas a Excel sin problemas.

Resultado Obtenido

El software se cierra abruptamente sin generar archivo y sin mostrar mensaje de error.

Representación del Error

Severidad: Alta

Este error afecta significativamente la funcionalidad del sistema y la productividad de los usuarios, impidiendo la exportación de grandes volúmenes de datos necesarios para reporting y análisis.
